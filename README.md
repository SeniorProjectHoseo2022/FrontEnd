# 2022.09-SeniorProject-Backend-node.js

졸업작품으로 진행된 프로젝트입니다.

구현된 기능은 다음과 같습니다.

1. 스팸 SMS와 유사도 테스트를 통해 스팸 메세지 구분
2. 스팸 SMS를 사용자에게 공지해주는 기능
3. 스팸 의심 SMS를 신고하고 설정된 횟수만큼 신고가 누적될 경우 블랙리스트에 등록되는 기능
